# Extended Use Case Description
### Use Case: _Log Results_

### Actors: _Registered User_

### Description:
1. The registered navigates to the Brew Result Logging page.
2. The user records brew data including tasting notes and the 5 point grading scale. 
3. The user submits the brew data which will be used in the statistics for the associated recipe and coffe.

### Cross Ref:
- Functional Requirements: 6 and 7
- Nonfunctional Requirements: 1, 4, and 7

### Requisite Use Cases: 
- User must have registered and logged in

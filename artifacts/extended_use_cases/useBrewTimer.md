# Extended Use Case Description
### Use Case: _Use Brew Timer_

### Actors: _Registered User_

### Description:
1. The registered navigates to the Recipe List screen. 
2. Once the user has created a recipe, they click on a recipe and select Use in Brew Timer.
3. User is routed to Brew Timer Page
4. User clicks Start button to begin timer

### Cross Ref:
- Functional Requirements: 16, 17, and 18
- Nonfunctional Requirements: 1, 4, and 8

### Requisite Use Cases: 
- User must have registered and logged in
- User must have create at least one recipe
